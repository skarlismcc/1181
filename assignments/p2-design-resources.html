Design Resources

Templates and Design Software

  • Canva
  • Adobe Express
  • Microsoft Word or Google Docs (best for simple, single-column designs, like blog articles)
  • Microsoft PowerPoint or Google Slides
    • Page size can be changes to be in portrait mode (i.e. 8.5x11" paper)
    • Text boxes, shapes, and pictures can be added and moved around into any layout.

Photos and Graphics

  • Unsplash: Free-to-use stock photos, icons, and illustrations
  • Pexels: Free-to-use stock photos
  • Pixabay: Free-to-use stock photos
  • Adobe Express: Good for icons and graphics
